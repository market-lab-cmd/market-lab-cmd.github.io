# Game Strategy Notes Hub

Root GitHub Pages site:

https://market-lab-cmd.github.io/

This repository is the main directory page for a network of educational game strategy notes. The homepage is written from a senior player's review perspective and links to Ludo, Teen Patti, Callbreak, Indian card game, and skill game study collections.

## Structure

- `index.html` - root GitHub Pages homepage
- `robots.txt` - crawl permission and sitemap location
- `sitemap.xml` - sitemap index for root pages and linked GitHub Pages collections
- `sitemaps/root-pages.xml` - root homepage and root review pages
- `sitemaps/collections.xml` - linked collection pages from the wider GitHub Pages network
- `content/` - Markdown source notes
- `assets/css/` - stylesheet files
- `assets/js/` - JavaScript files
- `assets/images/` - image assets for future root-page media
