# AGENTS.md

## Project Positioning

This repository is the root GitHub Pages index for the `market-lab-cmd.github.io` site network.

- Purpose: organize gameplay concepts, strategy notes, and review materials
- Voice: neutral, educational, senior-player review perspective
- Not allowed: promotional claims, guaranteed-win language, traffic manipulation wording

## URL

Root Pages URL:

https://market-lab-cmd.github.io/

## Directory Rules

- Homepage: `index.html`
- Markdown source notes: `content/`
- CSS: `assets/css/`
- JavaScript: `assets/js/`
- Images and future media: `assets/images/`
- Crawling files: `robots.txt`, `sitemap.xml`, `sitemaps/root-pages.xml`, `sitemaps/collections.xml`

## SEO Maintenance

- Keep one clear H1 on the homepage.
- Keep the canonical URL as `https://market-lab-cmd.github.io/`.
- Update `sitemap.xml` and the relevant file under `sitemaps/` when adding or removing public HTML pages.
- Keep page text in a senior-player review tone, not a marketing tone.
- Keep links to sub-sites accurate and crawlable.
