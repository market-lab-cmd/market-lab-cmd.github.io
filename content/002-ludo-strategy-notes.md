# Ludo Strategy Notes

Senior-player review angle: Ludo should be reviewed through position, spacing, token timing, safe squares, and whether a risk had a useful job.

Key questions:

- Was the captured token exposed for a reason?
- Did token spacing create choices or force weak moves?
- Was a safe square used to support progress or avoid playing?
- Did a capture improve the board or only answer frustration?
